Shortcodes Documentation
You can embed the plugin tables on any public page using Shortcodes.
How to add the table to a Page or Post
Go to the Pages or Posts section of your WordPress and create a new page.
If you use the new Block Editor (Gutenberg), click the [+] button to add a new block.
Search for the block named "Shortcode" and click it to insert it into your page.
Inside the text field of the block that appears, write exactly one of the shortcodes explained below.
Save or Update the page. Done! The site will replace that block with the actual table.
Note: if you use the old Classic Editor, you can just paste the shortcode directly in the middle of the rest of the text.
